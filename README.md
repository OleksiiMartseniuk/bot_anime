# Telegram bot 
Bot - interaction with the service [django_service_anime](https://github.com/OleksiiMartseniuk/django_service_anime)

#### Actions
* Receiving schedules
* Search by name
* Receiving anons
* Filter by genre

#### Technology
* Python => 3.10
* Aiogram
* Httpx

#### Instructions

Create a file at the root of the project `.env`

```
export TOKEN_BOT='you_token'
```edit
